Define los elementos que componen la estructuración de una página web

1. Contenedor (wrapper)

El contenedor o "wrapper" es un elemento que envuelve todo el contenido de una página web, proporcionando un ancho fijo o adaptativo para alinear y organizar el contenido.

2. Cabecera (header)

La cabecera o "header" es la sección superior de una página web que generalmente contiene el logotipo, el menú de navegación y otros elementos importantes.

3. Contenido (content)

El contenido o "content" es la parte principal de la página web donde se muestra la información y los elementos interactivos para el usuario.

4. Menú (menu)

El menú o "menu" es una sección que ofrece enlaces a diferentes páginas o secciones del sitio web, facilitando la navegación del usuario.

5. Pie (footer)

El pie o "footer" es la sección inferior de una página web que a menudo incluye información de contacto, enlaces legales y otros elementos de cierre.

6. Lateral (sidebar)

El lateral o "sidebar" es una columna ubicada a un lado del contenido principal que puede contener enlaces adicionales, información relevante o widgets.

7. Etiqueta <div>

La etiqueta <div> es un contenedor genérico en HTML que se utiliza para agrupar elementos y aplicar estilos o script mediante CSS y JavaScript.

Identifica y describe los elementos que componen la estructuración semántica de una página web

La estructuración semántica de una página web utiliza elementos HTML que proporcionan significado al contenido y mejoran la accesibilidad. Algunos de estos elementos incluyen:

1. <header>

Define la cabecera de una página o sección, típicamente utilizada para contener títulos, logotipos y menús de navegación.

2. <nav>

Define un bloque de navegación con enlaces a otras partes del sitio web.

3. <article>

Define un contenido independiente y autónomo dentro de una página, como un artículo de blog o una entrada de noticias.

4. <section>

Define una sección temática dentro de una página, usada para agrupar contenido relacionado.

5. <aside>

Define contenido secundario o complementario que puede ser tangencial al contenido principal, como barras laterales o notas adicionales.

6. <footer>

Define el pie de una página o sección, donde se colocan información de contacto, derechos de autor y enlaces de interés.
